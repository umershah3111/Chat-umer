# ChatGPT App Template

This is a Node.js/Express app with ChatGPT integration, ready for Render deployment.

## Setup

1. Push this repo to GitHub.
2. On Render, add an environment variable:
   - Name: OPENAI_API_KEY
   - Value: your OpenAI API key
3. Deploy the app on Render (New → Web Service → Connect GitHub Repo)
4. Build command: `npm install`
5. Start command: `npm start`

Visit the app URL and chat with ChatGPT directly!
